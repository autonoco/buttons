## Agent skill installer in buttons init

`buttons init` now sets up your coding agent to work with Buttons automatically.

### New features

- **Agent reference guide** — running `buttons init` now writes a `.buttons/AGENT.md` file that teaches any coding agent what Buttons is and how to use it. The file is always kept up to date when you re-run init.

- **Coding agent skill picker** — on an interactive terminal, `buttons init` prompts you to install a Buttons skill file for your preferred coding agent. Supported agents include Cursor, Claude Code, Cline, GitHub Copilot, and the cross-agent AGENTS.md convention. Select one or more to install, or skip to install nothing. See the [CLI reference](/cli/buttons_init).

- **Non-interactive agent flag** — pass `--agent cursor,claude` to install specific skill files without a prompt, or `--agent none` to skip the picker entirely. Useful for CI and scripted setups.

- **Idempotent updates** — re-running `buttons init` updates existing skill files in place rather than duplicating content, so you can safely re-run it after upgrading Buttons.
