## Agent skill picker in buttons init

`buttons init` now sets up your coding agent alongside your project.

### New features

- **AGENT.md reference guide** — running `buttons init` now writes a `.buttons/AGENT.md` file that teaches any coding agent what Buttons is and how to use it. The file is always kept up to date on each init. See [folder structure](/concepts/folder-structure).

- **Interactive skill picker** — on a TTY, `buttons init` prompts you to install a Buttons skill file for your coding agent. Supported agents include Cursor, Claude Code, Cline, GitHub Copilot, and a generic AGENTS.md. Nothing is installed unless you explicitly pick. See [skill.md](/ai/skill-md).

- **Non-interactive selection** — pass `--agent cursor,claude` to select agent integrations without a prompt, or `--agent none` to skip the picker entirely. Useful in scripts and CI.

- **Idempotent updates** — re-running `buttons init` replaces the Buttons section in place using start/end markers rather than duplicating content. Safe to run repeatedly.
</Update>
